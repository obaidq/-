# Abu Abed Box – Claude Code Starter Pack

This package turns your design research into a repo structure Claude Code can actually use.

## What is included

- `CLAUDE.md` → repo-wide instructions for Claude Code
- `docs/design-analysis.md` → design research summary
- `docs/ui-style-guide.md` → distilled visual direction
- `docs/ai-design-rules.md` → strict design rules for future generations
- `docs/design-system-template.md` → empty template Claude can fill
- `docs/master-prompts.md` → copy‑paste prompts for Claude Code
- `references/README.md` → where to place screenshots
