## 🚀 طريقة النشر (خطوة بخطوة)

### الخيار 1: النشر على Render.com (مجاني)

#### الخطوة 1: إنشاء حساب
1. اذهب إلى [render.com](https://render.com)
2. اضغط "Get Started"
3. سجل بحساب GitHub أو بإيميلك

#### الخطوة 2: رفع الكود على GitHub
1. أنشئ repository جديد على GitHub
2. ارفع ملفات المشروع (server.js, package.json, public/)

#### الخطوة 3: ربط المشروع بـ Render
1. في Render Dashboard، اضغط "New +"
2. اختر "Web Service"
3. اربط حسابك بـ GitHub
4. اختر الـ repository الخاص بالمشروع

#### الخطوة 4: إعدادات الخدمة
- **Name**: abu-abed-box
- **Runtime**: Node
- **Build Command**: `npm install`
- **Start Command**: `npm start`
- **Instance Type**: Free

#### الخطوة 5: انتظر النشر
- سيبدأ Render ببناء المشروع تلقائياً
- بعد دقائق، ستحصل على رابط مثل: `https://abu-abed-box.onrender.com`

---

### الخيار 2: النشر على Railway.app

```bash
# 1. ثبّت Railway CLI
npm install -g @railway/cli

# 2. سجل دخول
railway login

# 3. أنشئ مشروع جديد
railway init

# 4. انشر
railway up
```

---

### الخيار 3: النشر على Heroku

```bash
# 1. ثبّت Heroku CLI
# 2. سجل دخول
heroku login

# 3. أنشئ التطبيق
heroku create abu-abed-box

# 4. انشر
git push heroku main
```

---

### الخيار 4: التشغيل المحلي (للتطوير)

```bash
# 1. ثبّت المتطلبات
npm install

# 2. شغّل السيرفر
npm start

# 3. افتح المتصفح على
http://localhost:3000
```

## 💰 خطة الربح (Business Model)

### 1. نموذج Freemium
- **مجاني**: 3 ألعاب أساسية
- **Premium**: 10 ريال/شهر - كل الألعاب + بدون إعلانات

### 2. الإعلانات
- إعلانات بين الجولات
- رعايات من شركات سعودية

### 3. المشتريات داخل التطبيق
- حزم أسئلة إضافية (5-15 ريال)
- ثيمات وشخصيات (3-10 ريال)
- إزالة الإعلانات (20 ريال مرة واحدة)

### 4. الاشتراكات للشركات
- باقة الشركات: 500 ريال/شهر
- ألعاب Team Building مخصصة
- لوحة تحكم للمسؤولين

### 5. الفعاليات
- تنظيم بطولات برعاية
- شراكات مع المقاهي والمطاعم
